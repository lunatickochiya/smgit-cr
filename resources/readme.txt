Usage steps
1.Download and install smartgit from the official website
2.Download "crack SmartGit.jar" from Releases(https://gitee.com/pedoc/crackSmartGit/releases)
3.Modify the "smartgit.vmoptions"(%APPDATA%\syntevo\SmartGit\) file and add "-javaagent:C:\Users\Administrator\AppData\Roaming\syntevo\SmartGit\crackSmartGit.jar"(Please note that it must be an absolute path to the “crackSmartGit.jar” file) at the end
4.Put the target file in the smartgit installation directory(example:D:\Program Files\SmartGit\bin)
5.Modify the contents of the file to the actual installed version(default:18.2.5)
6.Start smartgit and select license.zip to register
7.Have fun
